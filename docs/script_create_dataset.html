script_create_dataset module¶

This script creates usable data sets for training.

Dependencies:
  • sox: this module uses the sox software to convert audio files to wav

format. Please be sure you have this software installed in your PATH system variable. More details in http://sox.sourceforge.net

Note:
Duplicate files are being ignored.
script_create_dataset.create_dataset(dataset_dir: str, file_list: list, num_workers: int = None, pre_processing: callable = None, **kwargs)[source]¶

Creates a dataset.

Parameters:
  • dataset_dir – str Output directory (data set).
  • file_list – list List of files to process (paths).
  • num_workers – int The maximum number of processes that can be used to execute the given calls
  • pre_processing – callable Pre process data before saving. Default to None, no pre processing will be performed.
  • kwargs – Additional kwargs are passed on to the pre processing function.
script_create_dataset.files_to_process(files_list: list, output_dir: str, contains_extension=True, is_base_name=True) → list[source]¶

Build a list of remaining files to process.

This function checks the data set integrity.

Parameters:
  • files_list – list The original list of file names to process. Only names is considered by default (is_base_name=True).
  • output_dir – str The path of the data set (output directory).
  • contains_extension – bool Must be True if the file names contains file extensions (for instance file.wav). Default to True.
  • is_base_name – bool Consider the provided list of files as a list of names. If you provide a list of paths, set this as False. Default to True.
Returns:

list A list containing the remaining files to process (in the format of the provided list).

Pre process a file. Use this function to handle raw data.
