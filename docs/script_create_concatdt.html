script_create_concatdt module¶

This script creates usable data sets for training. All data of the original bases will be concatenated before processing.

Dependencies:
  • sox: this module uses the sox software to convert audio files to wav

format. Please be sure you have this software installed in your PATH system variable. More details in http://sox.sourceforge.net

script_create_concatdt.trim(input_file: str, output_path: str, trim_interval: int = 5, num_workers: int = None, verbose_level=0)[source]¶

Trim an audio file.

This function will generated audio files with the original name of the file + position which the audio begins and the duration in seconds of the audio (for instance file_5.0_10.0.wav refers to an audio which original name is ‘file’, with 10 seconds of duration, and the start position begins at 5 seconds in the original file).

Parameters:
  • input_file – str Path to the file to process.
  • output_path – str Output path for the generated files.
  • trim_interval – tuple, shape=(start, end) Trim audio. Default to None (no trim will be performed).
  • num_workers – int The maximum number of processes that can be used to execute the tasks.
  • verbose_level – int Verbosity level. See sox for more information.
