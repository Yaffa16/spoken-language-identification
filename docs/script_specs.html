script_specs module¶

This script generates spectrogram of wav audio files using parallel processing.

script_specs.files_to_process(files_list: list, output_dir: str, contains_extension=True, is_base_name=True) → list[source]¶

Build a list of remaining files to process.

This function checks the output directories searching for generated spectrogram images.

Parameters:
  • files_list – list The original list of file names to process. Only names is considered by default (is_base_name=True).
  • output_dir – str The path of the data set (output directory).
  • contains_extension – bool Must be True if the file names contains file extensions (for instance file.wav). Default to True.
  • is_base_name – bool Consider the provided list of files as a list of names. If you provide a list of paths, set this as False. Default to True.
Returns:

list A list containing the remaining files to process (in the format of the provided list).
